About

This little app displays the latest track I have been listening to. I created it to get to know the MooTools JavaScript library, and also just for monkeying around with. I call it my My Last.fm / Deezer / MooTools Project.

Credits

All registered plays are tracked using the Last.fm AudioScrobbler. Most track plays are scrobbled from Deezer and soundcloud.com. Currently only working for my personal Last.fm account. The list updates itself every three minutes. Uses momentjs for date parsing. JavaScript concatenation done using UglifyJS2. Background image found on subtlepatterns.

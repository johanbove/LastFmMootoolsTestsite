About

This little app displays the latest ? tracks I have been listening to. The list updates itself every three minutes. I created it to learn the MooTools JavaScript library, and also for monkeying around with CSS3, Github and other tools. I call it my "Last.fm-Deezer-Scrobbling-MooTools" project.

Current version: v0.3.4

Credits

All registered plays are tracked using the Last.fm AudioScrobbler. Most track plays are scrobbled from Deezer and soundcloud.com. Uses momentjs for date parsing. JavaScript concatenation done using UglifyJS2. Background image found on subtlepatterns. Built with Microsoft's Webmatrix 3 which is a free and lightweight code editor which allows for running sites locally, including building and deploying your site.
